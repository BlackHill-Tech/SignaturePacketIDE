# SignaturePacketIDE

AI-powered legal signature page extractor and packet generator. Automates the creation of wet-ink signature packs for M&A and financing transactions.

## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   ```bash
   npm install
   ```

2. Set up environment variables:
   Create a `.env.local` file in the root directory and add your Gemini API key:
   ```
   API_KEY=your_api_key_here
   ```

3. Run the app:
   ```bash
   npm run dev
   ```
